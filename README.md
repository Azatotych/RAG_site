# Локальный ассистент (SPA)

Минимальный чат-UI на React + TypeScript + Vite для общения с локальным LLM-бэкендом. Сообщения хранятся только в памяти текущей вкладки. Поддерживаются светлая/тёмная темы с переключателем и сохранением выбора в `localStorage` (по умолчанию берётся системная тема).

Переключатель темы находится в правой части верхней панели рядом с кнопкой очистки.

Для оценки интерфейса можно запускать dev-сервер без настроенного API: UI отобразится, а в ленте появятся сообщения об ошибке подключения после каждого отправленного запроса.

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
```

## Переменные окружения

Настройте адрес API через `.env` или переменные среды:

```
VITE_API_BASE_URL=http://localhost:8080
VITE_CHAT_ENDPOINT=/api/chat
```

Итоговый запрос: `POST ${VITE_API_BASE_URL}${VITE_CHAT_ENDPOINT}` с телом:

```json
{
  "messages": [
    {"role":"user","content":"..."},
    {"role":"assistant","content":"..."}
  ],
  "stream": false
}
```

Ответ по умолчанию: `{ "reply": "текст ответа ассистента" }`.

Если переменные окружения не заданы или API недоступен (ошибка сети, DNS, HTTP-статус, таймаут, некорректный JSON/отсутствует `reply`), приложение продолжит работу: текст пользователя попадёт в ленту, а следом появится красное сообщение об ошибке с причиной и URL запроса.
